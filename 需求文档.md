# Paste Helper（粘贴助手）需求文档

> 版本：v1.0 | 更新日期：2026-07-15

---

## 1. 项目概述

### 1.1 项目名称

**Paste Helper** — Windows 桌面粘贴助手

### 1.2 项目定位

一款轻量级 Windows 桌面剪切板管理工具，让用户可以预存常用文本片段（命令、代码、模板、回复等），通过全局快捷键在任意应用中快速切换并粘贴。

### 1.3 目标用户

- 需要频繁粘贴重复文本的办公用户
- 需要在终端 / IDE 中快速输入常用命令的开发者
- 需要管理多段固定回复模板的客服 / 运营人员

## 2. 技术架构

### 2.1 技术选型

| 层级 | 技术 | 说明 |
|------|------|------|
| 桌面框架 | Wails v2 | Go 与 Web 前端的桥接框架 |
| 后端语言 | Go 1.25 | 业务逻辑、Win32 API 调用、数据持久化 |
| 前端框架 | Vue 3 (Composition API) | 响应式 UI |
| 构建工具 | Vite 7 | 前端构建 |

| 包管理 | Go modules + npm | 前后端各自管理依赖 |

### 2.3 数据存储

数据存储在 SQLite 数据库中，路径为 `%APPDATA%/paste-helper/data.db`，包含两张表：

**items 表** — 剪切板条目：

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT PK | UUID 唯一标识 |
| grp | TEXT | 分组（当前未使用，预留字段） |
| content | TEXT NOT NULL | 文本内容 |
| comment | TEXT | 注释说明 |
| created_at | INTEGER | 创建时间戳 |

**settings 表** — 应用设置（key-value 模式）：

| key | 默认值 | 说明 |
|-----|--------|------|
| current_index | -1 | 当前选中条目索引 |
| opacity | 85 | 窗口透明度 (10-100) |
| prev_keys | Ctrl,Up | 上一条快捷键 |
| next_keys | Ctrl,Down | 下一条快捷键 |
| theme | dark | 主题 (dark / light) |

## 3. 功能需求

### 3.1 窗口形态

| 需求项 | 说明 |
|--------|------|
| 位置 | 屏幕顶部居中 |
| 尺寸 | 横条：420 × 40 px；展开抽屉后：420 × 440 px |
| 边框 | 无边框（Frameless） |
| 置顶 | 始终置顶（AlwaysOnTop），不抢占焦点 |
| 透明度 | 可调节（10%-100%），通过 CSS `body.style.opacity` 实现 |
| 任务栏 | 不在任务栏显示（WS_EX_TOOLWINDOW） |

### 3.2 横条显示区（始终可见）

横条从左到右包含以下元素：

1. **◀ 按钮** — 切换到上一条，悬浮显示快捷键组合
2. **索引显示** — 格式 `3/10`，表示当前第几条 / 总共几条
3. **内容文本** — 显示当前条目内容（单行截断，溢出省略号）；无条目时显示 "暂无内容 — 双击添加"；有注释时在文本后追加 `· 注释内容`
4. **▶ 按钮** — 切换到下一条，悬浮显示快捷键组合
5. **▼ 按钮** — 展开 / 收起抽屉面板（展开时箭头旋转 180°）
6. **✕ 按钮** — 退出应用

**交互行为：**
- 双击横条 → 展开 / 收起抽屉
- 点击 ◀/▶ 按钮 → 切换条目并自动复制到剪切板
- 鼠标悬浮横条 → 显示提示条（完整内容 + 注释），移开 2.5s 后消失
- 导航切换后 → 自动弹出提示条持续 2.5s + Toast "已复制"

### 3.3 抽屉面板（展开时显示）

抽屉位于横条下方，包含两个标签页：

#### 3.3.1 条目标签页

- **添加表单**
  - 内容输入框（placeholder: "命令 / 文本内容..."）
  - 注释输入框（placeholder: "注释（可选）"）
  - 添加按钮（内容为空时禁用）
  - 支持回车键提交
- **条目列表**
  - 显示序号、内容文本、注释（若有）
  - 点击条目 → 选中并自动复制到剪切板
  - 悬浮时出现 ✕ 删除按钮
  - 当前选中条目高亮显示（左侧蓝色边框 + 蓝色背景）
  - 无条目时显示空状态提示 "还没有保存的条目"

#### 3.3.2 设置标签页

- **透明度调节** — 滑块范围 10%-100%，显示当前百分比，150ms 防抖
- **上一条快捷键** — 点击激活捕获模式，按下组合键后保存（需包含至少一个修饰键 Ctrl / Alt / Shift / Win + 一个普通键）
- **下一条快捷键** — 同上
- **退出按钮** — 位于抽屉底部

### 3.4 全局快捷键

| 功能 | 默认快捷键 | 说明 |
|------|-----------|------|
| 上一条 | Ctrl + Up | 切换到上一个条目并复制到剪切板 |
| 下一条 | Ctrl + Down | 切换到下一个条目并复制到剪切板 |

- 快捷键通过 Win32 `RegisterHotKey` API 实现全局注册
- 循环切换：最后一条 → 下一条 → 第一条，第一条 → 上一条 → 最后一条
- 用户在设置中可自定义快捷键
- 快捷键修改后实时更新注册（通过 channel 通知消息泵重新注册）

### 3.5 剪切板操作

- 条目切换时**自动复制**内容到系统剪切板（UTF-16 格式，Win32 API）
- 复制通过 `OpenClipboard` → `EmptyClipboard` → `SetClipboardData` → `CloseClipboard` 实现

### 3.6 主题

- 深色主题（默认）/ 浅色主题
- 通过 CSS 变量切换，`data-theme` 属性控制

### 3.7 系统托盘（待集成）

tray 模块已实现以下功能，尚未接入 `main.go`：

## 4. 非功能需求

### 4.1 性能

- 应用启动时间 < 1s
- 快捷键响应延迟 < 100ms
- 数据库操作同步执行（SQLite 单连接模式），无并发问题

### 4.2 兼容性

- 操作系统：Windows 10+
- 依赖：WebView2 运行时（Windows 10 1809+ 已内置）

### 4.3 构建

```bash
# 开发
wails dev

# 生产构建
wails build
```

构建产物：`paste_helper.exe`
